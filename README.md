# Frontend GH Readme

## Процесс разработки

### Создание веток

В этом проекте используется автоматическое создание веток через GitHub Actions.

#### Как это работает:

1. **Создайте Issue** в GitHub с описанием задачи
2. **Система автоматически** создаст ветку в формате: `issue/{номер-issue}`
3. **Для начала работы** переключитесь на созданную ветку:
   ```bash
   git fetch --all
   git checkout issue/{номер-issue}
4. **Коммиты** создаем коммит с указанием номера issue
  ```bash
    git commit -m "#{номер-issue} текст коммита"
